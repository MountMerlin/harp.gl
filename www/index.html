xyz:three.js
free and open source 3d web map rendering engine
powered by three.js

Web first

xyz:three.js is a flexible map renderer powered by modern web technologies like WebGL and three.js.

Open source

xyz:three.js is a fork of now disfunct harp.gl codebase with proprietary features and gating APIs removed.

Flexible

Add any type of datasource such as GeoJSON, vector and raster tiles, or custom objects to the map.
